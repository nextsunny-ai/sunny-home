키는 반드시 sk-ant-api03-로 시작 (= OAuth 토큰 X). 비용 = 본인 키 본인 차감 = 종량제.

GitHub 연결

할 일·일정을 PC 5대·폰에서 똑같이 보려면 GitHub 토큰을 한 번만 등록하면 됩니다. 등록 후에는 입력하는 즉시 자동 저장됩니다 — 따로 "푸시"할 필요 없습니다.

토큰 만드는 법 (1분)
1. github.com/settings/personal-access-tokens 열기
2. Repository access → Only select repositories → sunny-home
3. Permissions → Contents 를 Read and write 로
4. 만들어진 github_pat_… 토큰을 아래에 붙여넣기
