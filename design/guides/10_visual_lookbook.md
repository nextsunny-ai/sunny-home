# Visual Lookbook — 디자인 가이드

이 파일을 Claude 또는 ChatGPT의 **Project Knowledge**에 업로드한 뒤, 그 프로젝트에서 PPT 콘텐츠 생성을 요청하면 이 톤이 일관되게 적용된다.

하이패션 매거진(Vogue·Harper's Bazaar 계열)의 에디토리얼 문법을 따른다. 핵심 원리 두 가지 — **사진이 지면을 지배하고, 디돈(Didone) 세리프 타이포가 사진 위의 "베일"처럼 얹힌다.** 텍스트가 사진과 경쟁하기 시작하면 이 스타일이 아니다.

---

## 폰트

- **디스플레이·폴리오** = Bodoni Moda (디돈 계열 고대비 세리프 · 이탤릭 혼용)
- **한국어** = Noto Serif KR (명조 — 고딕 금지)
- 모노스페이스 폰트 = 이 스타일에서 사용 금지 (보고서 톤이 됨)
- system-ui / Inter / Roboto / Pretendard 등 산세리프 = 사용 금지
- Google Fonts CDN `@import` 사용

## 컬러

```
페이퍼 프레임   #F3EFE7  (크림 페이퍼)
잉크           #221C16  (페이퍼 위 텍스트)
사진 위 타이포  #FBF7EE  (웜 화이트)
폴리오·크레딧   rgba(251, 247, 238, 0.75~0.9)
```

- 별도 액센트 컬러 없음 — 색은 사진이 가져오고, 타이포는 웜 화이트·잉크 2색뿐
- **`#FFFFFF` / `#000000` 절대 X**
- 사진 위 컬러 텍스트·박스 X. 타이포의 대비는 크기와 세리프의 굵은-가는 획 대비로 만든다

## 이미지 처리 룰 (= 이 스타일의 핵심)

1. **지면 배분** — 페이지의 85% 이상이 사진. 텍스트는 마스트헤드·폴리오·크레딧·캡션 1줄까지만
2. **페이퍼 프레임** — 모든 페이지는 26px 크림 페이퍼 보더 안에 사진이 앉는다 (인쇄 화보의 마진 감각). 프레임 없는 변형은 풀블리드 1장에만 허용
3. **타이포 베일** — 헤드라인은 사진 위에 직접 얹는다. 스크림·반투명 박스 X — 대신 사진의 밝은 영역(하늘·여백) 위에 배치하고, 위치가 안 나오면 타이포를 옮긴다 (사진을 보정하지 않는다)
4. **시리즈 톤 통일** — 한 룩북 안의 사진은 같은 필름톤(웜·뉴트럴·모노 중 1개)으로 보정 통일
5. **이미지 비율** — 풀블리드 16:9 · 프레임 내 3:2 · 세로컷 4:5. 한 페이지 1장이 기본, 병치는 2장까지
6. **크레딧 의무** — 사진마다 이탤릭 크레딧 (`ph. 장소 또는 작가, 연도`) 우하단
7. **placeholder** — 실제 사진이 없으면 필름톤 다층 그라데이션 + 실루엣(능선·인물·오브제)으로 회화적으로 구성하고 크레딧 자리에 `ph. TBD` 표기. 단색 박스·빗금 박스로 때우지 않는다

## 레이아웃

- 슬라이드 = **1920 × 1080** · 페이퍼 패딩 26px
- 마스트헤드 = 상단 중앙 정렬: 이슈 라인 (레터스페이스 대문자) → 디돈 헤드라인 → 한 줄 덱 (Noto Serif KR · 자간 넓게)
- 폴리오 = 하단 중앙 `N° 01 — 12` (Bodoni Moda · 자간 0.3em)
- 크레딧 = 우하단 이탤릭
- 그리드·칩·hairline·카드 구조 = 사용 금지 (보고서 문법)

## 타이포

| 요소 | 폰트 | 크기 | 비고 |
|---|---|---|---|
| 마스트헤드 | Bodoni Moda 500 | 160~200px | 이탤릭 단어 혼용 · line-height 0.9 |
| 이슈 라인 | Bodoni Moda 500 | 15~16px | uppercase · letter-spacing 0.42em |
| 덱 (한국어) | Noto Serif KR 400 | 18~20px | letter-spacing 0.3em |
| 본문 캡션 | Noto Serif KR 400 | 15~16px | line-height 1.8 · 최대 2문장 |
| 폴리오·크레딧 | Bodoni Moda | 14~15px | 크레딧은 이탤릭 |

## 페이지 5장 패턴

1. **커버 (Cover)** — 풀블리드 사진 + 디돈 마스트헤드 (이슈 라인·헤드라인·덱) + 폴리오·크레딧
2. **오프닝 스프레드 (Opening)** — 페이퍼 프레임 안 사진 1장 + 좌상단 디돈 챕터 번호 (`I.`) + 캡션 1줄
3. **풀블리드 (Plate)** — 사진 한 장. 텍스트 = 폴리오 + 크레딧뿐
4. **병치 (Diptych)** — 프레임 안 사진 2장 나란히 (같은 톤) + 각 1줄 캡션
5. **콜로폰 (Colophon)** — 사진 없는 페이퍼 페이지. 디돈 대형 `Fin.` + 크레딧 목록 (Photography·Art Direction·Contact)

## 금기 (절대 X)

- 모노스페이스·고딕 폰트 (이 스타일은 세리프 전용)
- 스크림 박스·반투명 패널 위 텍스트 (타이포는 사진 위에 직접)
- 그리드 카드·칩·hairline 메타 바 (보고서 문법)
- 이미지 위 그라디언트 오버레이
- 이모지 · 둥근 카드 · 액센트 컬러
- 영업·홍보·광고 톤 문구 ("Powerful·Premium·혁신적·새로운 ___을 만나다")
- AI 슬롭 패턴 (보라→핑크 그라디언트 · 글로우 · 무지개 팔레트)
- 사람 얼굴 stock 사진 (실루엣·뒷모습·풍경으로)

## 출력 형식

- HTML = `<section class="slide">` vertically stacked · CSS `<style>` 임베디드
- 폰트 = Google Fonts `@import` (Bodoni Moda · Noto Serif KR)
- 실제 사진 = `<img>` 또는 `background-image` + `object-fit: cover`
- 콘텐츠(주제·카피)는 사용자 주제대로

## 자가 검토

- [ ] 타이포가 사진 위에 직접 (스크림·박스 없음)
