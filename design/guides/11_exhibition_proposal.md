# Exhibition Proposal — 디자인 가이드

이 파일을 Claude 또는 ChatGPT의 **Project Knowledge**에 업로드한 뒤, 그 프로젝트에서 PPT 콘텐츠 생성을 요청하면 이 톤이 일관되게 적용된다.

미술관·비엔날레 전시 카탈로그의 문법을 따른다 — 스위스 국제 타이포그래피 양식(기하 그리드 · 객관적 · 서체 2종 제한)에, 전시 도록의 관행인 **Fig. 번호 체계**와 **tombstone(작품 라벨: 명칭·치수·연도)** 표기를 결합한다. 클라이언트가 공간을 "아카이브된 작품"처럼 신뢰하며 읽게 만드는 것이 목적이다.

## 폰트

- **라틴·숫자·라벨** = Space Grotesk (그로테스크 — 폴리오·Fig. 번호·메타)
- **한국어** = Noto Sans KR (400 / 500 / 700 / 900)
- 서체 = 정확히 2종. 세리프·모노스페이스 추가 금지
- system-ui / Inter / Roboto / Pretendard 사용 금지
- Google Fonts CDN `@import` 사용

## 컬러

```
배경        #F7F6F2  (페이퍼 화이트)
잉크        #141310
액센트      #A8431E  (Rust — 폴리오·Fig. 번호에만)
구조 룰     #141310 솔리드 1~2px (잉크 풀톤 — 연한 hairline 금지)
muted       rgba(20, 19, 16, 0.5~0.7)
도면 바탕   #FCFBF8 + 모눈 rgba(20,19,16,0.06)
```

- **`#FFFFFF` / `#000000` 절대 X**
- 구분선은 잉크색 솔리드 룰 — 스위스 문법은 연한 회색 hairline이 아니라 명확한 검정 룰을 쓴다
- 액센트는 rust 1색, 번호 체계에만. 제목·본문에 컬러 X. 존 구분 무지개 컬러 X

## 이미지 처리 룰 (= 이 스타일의 핵심)

1. **Fig. 번호 체계** — 모든 시각 자료(사진·렌더·도면)에 `Fig. 01`부터 일련번호. 라벨은 이미지 **밖** 상단 라벨 바에 (이미지 위 오버레이 X). 라벨 바 = `Fig. NN — 명칭` 좌측 · 수치(면적·축척) 우측
2. **도면·사진 병치** — 같은 공간의 도면과 사진(렌더)을 한 페이지에 나란히 — 도면이 증거, 사진이 체감. 도면은 항상 축척 표기 (`1:200`)
3. **tombstone 표기** — 텍스트 컬럼 하단에 작품 라벨식 표: `Fig. NN / 명칭 — 내용 · 수치`. 면적은 `m²`, 치수는 mm 단위
4. **지면 배분** — 시각 자료 60% 이상 + 텍스트 컬럼 1개. 텍스트 컬럼이 2개를 넘으면 보고서가 된다
5. **이미지 톤** — 출처가 다른 레퍼런스와 자체 렌더를 한 페이지에 섞을 때는 같은 명도·채도 범위로 보정. 어려우면 페이지 분리
6. **placeholder** — 사진이 없으면 공간 분위기를 회화적으로 구성(아치 회랑·발광 스크린·관람객 실루엣 등 다층 그라데이션 + clip-path)하고 Fig. 라벨은 그대로. 도면 placeholder = 모눈 + 2px 잉크 선 룸 윤곽

## 레이아웃

- 슬라이드 = **1920 × 1080**
- 기본 그리드 = **좌 텍스트 컬럼 4 : 우 시각 자료 8** (12컬럼 기준). 컬럼 경계 = 2px 잉크 룰
- 텍스트 컬럼 구조 = 상단 아카이브 라벨 블록 (`Exh. N° / Venue / Period`) → 중앙 대형 폴리오 숫자(rust) + 국문 제목(Noto Sans KR 900) + 영문 캡스 라인 → 하단 tombstone 표
- 시각 자료 영역 = 와이드 1장 + 하단 2분할 (또는 2×2). 셀 경계 = 1px 잉크 룰
- 둥근 모서리·그림자·칩 = 사용 금지. 모든 요소가 직각 그리드 위에

## 페이지 5장 패턴

1. **커버 (Cover)** — 좌 텍스트 컬럼 (아카이브 라벨 + 폴리오 + 제목 + tombstone) / 우 와이드 사진 1 + 도면·사진 2분할
2. **평면 계획 (Plan)** — 도면 크게 (Fig. + 1:200) + 우측 존 tombstone 목록 (ZONE · 명칭 · 면적)
3. **공간 연출 (Views)** — 사진·렌더 2×2 병치, 각각 Fig. 라벨 바
4. **동선·운영 (Circulation)** — 도면 + 동선 포인트 번호 목록 (스위스 넘버링 01·02·03)
5. **마무리 (Imprint)** — 텍스트만: 일정·예산 범위·다음 단계 표 + 연락처 (도록 판권면 문법)

실제 제안서는 존 개수만큼 패턴 2·3을 반복 확장한다.

## 금기 (절대 X)

- 서체 3종 이상 · 모노스페이스 · 세리프
- 연한 회색 hairline (구분선은 잉크 솔리드 룰)
- 이미지 위 텍스트 오버레이 (라벨은 이미지 밖 라벨 바)
- 존 구분 무지개 컬러 (rust 1색 + 번호로)
- 둥근 카드 · 그림자 · 칩 · 그라디언트
- 이모지
- 영업·홍보·광고 톤 문구 ("Powerful·Premium·혁신적·새로운 ___을 만나다")
- AI 슬롭 패턴 (보라→핑크 그라디언트 · 글로우)
- Fig. 번호 없는 시각 자료
