# Global Slide Tools — 통합 가이드

7개 슬라이드 도구의 특징 · 사용 시점 · 무료/유료. AI에 직접 업로드하면 = "Pitch 스타일로" 식 호출. 또는 = 직접 도구를 사용할 때 의사결정 참고.

---

## 1. Pitch

URL = https://pitch.com / 한국어 X · 영어
- **무료 + 유료 ($8/월~)**
- 협업형 = 동시 편집 · 댓글 · 분석
- 템플릿 풍부 + 모던 톤
- 브랜드 키트 + 폰트 임포트

용도: 팀 협업 슬라이드 · IR · 영업 자료

## 2. Gamma

URL = https://gamma.app
- **AI 기반** = 주제만 박으면 자동 생성
- PPT + 웹페이지 + 문서 출력 다 OK
- 무료 한도 + Pro $10/월
- 한국어 지원

용도: 빠른 초안 + AI 자동 생성

## 3. Beautiful.ai

URL = https://www.beautiful.ai
- **AI 자동 레이아웃**
- 슬라이드 타입별 템플릿 (= 차트·비교·타임라인)
- 무료 한도 + Pro $12/월
- 미국 톤 비즈니스

용도: 비즈니스 피치덱 + 자동 정렬

## 4. Slidesgo

URL = https://slidesgo.com
- **무료 PPT 템플릿** = Google Slides · PowerPoint
- 다양한 카테고리 (교육·비즈니스·의료 등)
- 한국어 호환

용도: 무료 템플릿 다운 + 본인이 직접 편집

## 5. Canva Presentations

URL = https://www.canva.com/presentations
- **무료 + Pro $13/월**
- 가장 광범위한 템플릿 + 자산
- 비디자이너 친화적
- 협업 + 영상·이미지 통합

용도: 비디자이너 · 마케팅팀 · 빠른 출력

## 6. Pages.xyz

URL = https://pages.xyz
- 랜딩 페이지 · 단일 페이지 디자인 영감
- 슬라이드는 아니지만 = **컴포지션 레퍼런스로 유용**
- 무료 (큐레이션 사이트)

용도: 표지·히어로 레이아웃 영감

## 7. Slidebean

URL = https://slidebean.com
- **IR · 피치덱 전문**
- AI 자동 생성 + 컨설팅
- 유료 = $24/월~

용도: IR · 투자자 피칭 전문 슬라이드

## 사용 시점 가이드

| 상황 | 추천 도구 |
|---|---|
| 빠른 초안 (= AI 자동) | Gamma → Beautiful.ai |
| 협업·팀 발표 | Pitch → Canva |
| IR · 투자자 | Slidebean → Pitch |
| 비디자이너 사용 | Canva → Slidesgo |
| 디자인 자체 제어 | 직접 HTML (= 이 갤러리의 가이드) |
