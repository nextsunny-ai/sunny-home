# Atlassian Design — 디자인 가이드

## 핵심 원칙
- **워크 프로덕트 톤** (= 정직한 비즈니스 · 화려함 X)
- 블루 액센트 단색
- 데이터 + 협업 중심
- Confluence · Jira · Trello 통합 디자인

## 컬러
- Primary = `#0052CC` Atlassian Blue
- Background = `#FFFFFF` / `#F4F5F7`
- 텍스트 = `#172B4D`
- 액센트 컬러들 = Red · Yellow · Green (= 상태 표시용)

## 타이포
- Atlas Grotesk (대체 = Inter · Pretendard)
- weight 400 · 500 · 700

## 레이아웃
- 정직한 카드 + 그리드
- 라운드 = 3~4px (= 거의 직각)
- 헤더·사이드바·메인 3열

## 사용 시점
- 사내 협업 자료
- 프로젝트 관리 · 일정 발표
- 비즈니스 도구 데모
