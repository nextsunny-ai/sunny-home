# Pinterest — 영감 가이드

## 무엇
- 핀터레스트 = 이미지 보드 + 큐레이션
- "Presentation Design" 검색 = 다양한 PPT 시안 핀
- 본인 보드 만들어서 = 자료별 무드 정리

## 사용 시점
- **본인 큐레이션 보드** 만들기
- 자료 종류별 (= IR · 매거진 · 페스티벌) 보드 분리
- 캡처해서 다운로드 → AI에 첨부

## 보드 만들기 권장
1. 본인 핀터레스트 계정 만들기
2. 자료별 보드 5~10개 (= 회사·자료 종류별)
3. 좋은 핀 = 보드에 저장
4. 자료 만들 때 = 그 보드 보면서 = AI에 캡처 첨부

## 슬라이드 적용
5~10장 캡처 → AI에:
"이 톤들을 종합해서 PPT 5장을 HTML로 만들어 줘. 컬러·폰트·레이아웃 키워드 = {본인 강조}"

## 한계
- 이미지 저작권 = 외부 공유 X
- 직접 따라 만들기 = OK (= 톤만 가져옴)
