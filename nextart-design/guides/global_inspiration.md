# Global Inspiration — 통합 가이드

8개 큐레이션 사이트의 특징 · 사용 시점. PPT 톤·레이아웃·비주얼 영감 참고용.

---

## 1. Awwwards

URL = https://www.awwwards.com
- 웹 디자인 시상식 · 매일 1개 Site of the Day 발표
- **2025 키워드**: 3D 인터랙션 · 커스텀 커서 · 키네틱 타이포 · 풀스크린 영상
- 다크 모드 비중 70%+

용도: 임팩트 표지 · 키네틱 타이포 영감

## 2. Dribbble

URL = https://dribbble.com
- UI · 일러스트 · 비주얼 디자인 샷
- **검색 키워드**: `editorial web design` · `mesh gradient UI` · `bento grid` · `brutalist web design`

용도: 단일 컴포넌트·디테일 영감

## 3. Behance

URL = https://www.behance.net
- Adobe 운영 · 풀 프로젝트 (브랜딩 · 에디토리얼)
- **포트폴리오 풀 케이스** 보기에 최적

용도: 브랜드 전체 아이덴티티 · 매거진 톤 영감

## 4. Mobbin

URL = https://mobbin.com
- **모바일 앱 UI 패턴 라이브러리**
- 실제 앱 스크린샷 카테고리화
- 무료 한도 (= 최근 4개만) / Pro 유료

용도: 모바일 UI · 패턴 · 인터랙션 영감

## 5. Land-book

URL = https://land-book.com
- 랜딩 페이지 큐레이션 갤러리
- 카테고리 + 컬러 필터

용도: 랜딩 페이지·표지 영감

## 6. SiteInspire

URL = https://www.siteinspire.com
- 엄선된 웹 디자인 갤러리
- 스타일·subject·type별 필터

용도: 톤·스타일 비교 검토

## 7. Godly

URL = https://godly.website
- **극상위 웹 디자인 큐레이션**
- "신의 작품"이라고 부를 만한 = 가장 정교한 사이트들

용도: 최상위 톤 영감 (= 야망 있는 자료 만들 때)

## 8. Pinterest · Presentation Design

URL = https://www.pinterest.com/search/pins/?q=presentation%20design
- PPT · 키비주얼 보드 검색
- 핀터레스트 보드 만들어서 본인 컬렉션

용도: 직접 보드 만들어서 자료별 무드 정리

---

## 큐레이션 사용 방법

1. **본인 작업 카테고리 정의** = "IR" / "페스티벌 키비주얼" / "내부 보고서"
2. **2~3 사이트 골라서 큐레이션** = Pinterest 보드 또는 본인 다운로드 폴더
3. **5~10장 캡처** = 본인이 좋다고 느낀 톤만
4. **그 캡처들을 AI에 넘기기** = "이 톤을 따라 PPT 5장 HTML로"
