# Linear Method — 디자인 가이드

## 핵심 원칙
- **극도 미니멀** — 불필요한 UI 요소 0
- **키보드 퍼스트** — 모든 액션 단축키 (Cmd K · J · K)
- 60fps 부드러운 모션 (페이지 이동 · 모달 · 드롭다운)
- 다크 모드 디폴트 (= 개발자 타깃)

## 컬러
- 배경 = `#08090A` (= 거의 검정이지만 약간 따뜻)
- 텍스트 = 뉴트럴 그레이 다층
- 액센트 = 퍼플 `#5E6AD2`
- 텍스트 계층 = primary / secondary / tertiary 3단계

## 타이포
- 자체 Linear Sans (= 대체 = Inter)
- weight 400 · 500 · 600
- letter-spacing 0

## 레이아웃
- 사이드바 + 메인 패널 + 디테일 패널 = 3열
- 컴포넌트 간격 = 8 grid
- 라운드 = 4 · 6 · 8

## 사용 시점
- 프로덕트 데모
- 개발자 도구 자료
- 다크 모드 + 미니멀
