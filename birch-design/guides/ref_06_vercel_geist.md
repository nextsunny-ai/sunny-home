# Vercel Geist — 디자인 가이드

## 핵심 원칙
- **Geist Sans / Geist Mono** 폰트 (= Vercel 공식)
- 미니멀 흑백 + 단일 액센트 (= 보통 블루 또는 퍼플 그라디언트)
- CSS 변수 기반 토큰 (spacing · color · typography 모두)
- Next.js + App Router 기반

## 컬러
- Background = `#FFFFFF` (라이트) · `#000000` (다크)
- 텍스트 = 흑/백 + 회색 다층
- 액센트 = `#0070F3` Vercel Blue (그라디언트로도)
- 카드 = `#FAFAFA` 또는 `#111111`

## 타이포
- Geist Sans = 산세리프 깨끗한
- Geist Mono = 모노스페이스
- weight 100~900 variable

## 레이아웃
- 큰 여백 + 정렬된 컴포넌트
- 카드 라운드 8~12px
- 헤어라인 = 1px 회색

## 사용 시점
- Tech · SaaS · 기술 자료
- 개발자 도구 발표
- 모던 미니멀 (= 한국 카카오·토스도 비슷)
