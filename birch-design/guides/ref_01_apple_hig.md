# Apple HIG — 디자인 가이드

## 핵심 원칙
- **벤토 그리드** — 큰 카드 1 + 작은 카드 N의 불균등 배치
- 라운드 16~24px (= Apple 시그니처. 다른 톤에서는 둥근 카드 X)
- 큰 여백 + 단색 카드 + 카드별 컬러 1개
- SF Pro 폰트 (대체 = Geist / Inter)

## 컬러
- 라이트 = `#FFFFFF` 베이스 + 텍스트 `#000000`
- 다크 = `#000000` 베이스 + 텍스트 `#FFFFFF`
- 시스템 컬러 = `#007AFF` 블루 · `#34C759` 그린 · `#FF3B30` 레드 등
- 카드별 액센트 = 1개 hue

## 타이포
- Title = SF Pro Display · weight 700 · letter-spacing -0.02em
- Body = SF Pro Text · weight 400 · line-height 1.5

## 사용 시점
- 제품 쇼케이스
- IP 포트폴리오
- 모바일 앱 발표
- 미니멀 + 프리미엄

## 출력 형식
1920×1080 HTML 5장 stacked. 벤토 그리드 표지 + 카드 콘텐츠.
