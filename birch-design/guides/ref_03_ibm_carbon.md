# IBM Carbon — 디자인 가이드

## 핵심 원칙
- **엔터프라이즈 톤** — 회색 다층 + 파란 액센트 단색
- **직각 카드** (= 라운드 X)
- 데이터 시각화 강함 (= 차트·테이블·grid)
- WCAG AAA 접근성

## 컬러
- Primary = `#0F62FE` IBM 블루
- Gray 단계 = `#161616` · `#262626` · `#393939` · ... `#F4F4F4`
- 라이트/다크 두 테마
- 액센트 1개만 (= IBM 블루)

## 타이포
- IBM Plex Sans (또는 = Inter 대체)
- IBM Plex Mono = 코드·데이터
- letter-spacing 0

## 레이아웃
- 12 컬럼 grid · 16px gutter
- spacing = 4 · 8 · 16 · 24 · 32 · 48 · 64 px (= 8 grid)
- elevation = `box-shadow` 4단계

## 사용 시점
- B2B 엔터프라이즈 자료
- 데이터 리포트 · 차트 중심
- 정부·금융·산업 자료
