# Dribbble — 영감 가이드

## 무엇
- 디자인 샷 (= 작은 이미지) 모음 사이트
- UI · 일러스트 · 비주얼 디자인
- 디자이너 포트폴리오 + 채용

## 검색 키워드
- `editorial web design` = 잡지 스타일 비대칭 그리드
- `mesh gradient UI` = 그라디언트 메시
- `bento grid` = Apple 벤토
- `brutalist web design` = 브루탈리즘
- `minimal whitespace` = 미니멀 화이트
- `retro futurism web` = 레트로 퓨처리즘
- `organic shapes UI` = 둥근 blob

## 사용 시점
- 단일 컴포넌트 영감 (= 카드 1장의 디테일)
- 일러스트 · 아이콘 스타일
- 컬러 팔레트 큐레이션

## 슬라이드 적용
샷 5~10개 캡처 → AI에:
"이 디자인 톤으로 PPT 5장을 HTML로 만들어 줘."
