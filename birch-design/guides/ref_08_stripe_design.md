# Stripe Design — 디자인 가이드

## 핵심 원칙
- **그라디언트 메시 배경** = Stripe 시그니처 (블루·퍼플·핑크·시안)
- 마이크로 인터랙션 정교 (= 버튼·카드·탭 모두)
- 인디고·퍼플 + 화이트 + 뉴트럴
- 카드 = 12~16px 라운드 + 서브틀 섀도우
- 코드 블록 + 인터랙티브 예제 (= 문서 디자인 골드 스탠다드)

## 컬러
- Primary = `#635BFF` 인디고
- Background = `#FFFFFF`
- 그라디언트 = `linear-gradient(135deg, #635BFF 0%, #00D4FF 50%, #FF6BC8 100%)`
- 텍스트 = `#0A2540` 다크 블루
- 카드 = 흰색 + `box-shadow: 0 2px 8px rgba(0,0,0,0.04)`

## 타이포
- 자체 Sohne (= 대체 = Inter · Pretendard)
- weight 400 · 500 · 600 · 700
- 코드 = 모노스페이스 분리

## 사용 시점
- IR · 결제 · B2B 프리미엄
- API · 개발자 자료
- 그라디언트 임팩트 필요한 표지
