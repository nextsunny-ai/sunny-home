# Godly — 영감 가이드

## 무엇
- **극상위 웹 디자인 큐레이션**
- "신의 작품"이라고 부를 만한 = 가장 정교한 사이트들
- 매우 selective (= 큐레이션 엄격)

## 사용 시점
- **최상위 톤 영감** = 야망 있는 자료 만들 때
- 디테일 · 모션 · 타이포 = 극상위
- 본인 자료가 = "이 정도면 충분"이라 만족 X 일 때

## 한계
- 따라 만들기 = 시간·기술 많이 필요
- 영감으로만 = 적정

## 슬라이드 적용

"이 톤은 매우 정교한 디테일. 본인 자료는 = 이 톤의 정수 (= 미니멀 + 한 가지 임팩트) 만 가져와서 PPT 5장 HTML로."
