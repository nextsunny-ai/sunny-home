# Material 3 — 디자인 가이드

## 핵심 원칙
- **Material You** — 사용자 배경 색에서 톤 자동 추출
- elevation 시스템 5단계 (0 · 1 · 3 · 6 · 8 · 12)
- 부드러운 라운드 (12 · 16 · 28)
- 동적 컬러 + 표면 컬러

## 컬러
- Primary · Secondary · Tertiary 3 hue
- 각 hue = light / dark 톤 변주
- Surface = bg · variant
- 자동 색 추출 시스템

## 타이포
- Roboto Flex (또는 = Pretendard 대체)
- Display · Headline · Title · Body · Label 5단계
- 각 단계 = Large · Medium · Small 3변주

## 모션
- `cubic-bezier(0.2, 0, 0, 1)` standard easing
- duration = 100 · 200 · 300 · 500ms

## 사용 시점
- Android 호환 제품
- 안드로이드 앱 데모
- 부드러운 모션 강조
