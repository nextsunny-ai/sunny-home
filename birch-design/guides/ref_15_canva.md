# Canva Presentations — 슬라이드 도구 가이드

## 무엇
- 가장 광범위한 디자인 도구
- 슬라이드·소셜·문서·영상 다
- 비디자이너 친화적

## 가격
- 무료 (= 기본 템플릿)
- Pro $13/월 = 모든 템플릿 + 자산 + 브랜드 키트
- Teams = 협업 강화

## 장점
- 템플릿 압도적 양
- 사진·아이콘·일러스트 자산 통합
- 한국어 지원
- 비디자이너도 OK

## 단점
- 톤 = Canva 특유 = 약간 일반적
- 디자인 자유도 = 자동 vs 자유 사이

## 사용 시점
- 비디자이너 · 마케팅팀
- 빠른 출력 (= 영상·이미지 통합)
- 학교·교회·이벤트 자료
