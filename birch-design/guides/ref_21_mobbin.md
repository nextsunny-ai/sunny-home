# Mobbin — 영감 가이드

## 무엇
- **모바일 앱 UI 패턴 라이브러리**
- 실제 앱 스크린샷 카테고리화
- iOS · Android · Web (최근 추가)

## 가격
- **무료 한도** = 최근 4개 사이트만
- Pro = 유료 (= 본인이 가입해야 풀 사용)

## 사용 시점
- 모바일 UI 패턴 참고
- 인터랙션 · 온보딩 · 결제 등 = 카테고리별 검색
- 앱 데모 슬라이드 시 = 실제 앱 캡처 비슷한 톤

## 슬라이드 적용
- 우리 갤러리도 = Mobbin 패턴 영향 (= 3열 카드 + 회색 박스 안 흰색 미리보기)
- 앱 UI 슬라이드 만들 때 = Mobbin 카테고리 보고 톤 결정
